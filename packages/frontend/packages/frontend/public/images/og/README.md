# Open Graph Images

This directory contains Open Graph images for social media sharing.

## Required Images

1. **homepage.jpg** (1200x630px)
   - Hero image with Spanish Class branding
   - Spanish red gradient background
   - "Premium Spanish Classes Online" text

2. **about.jpg** (1200x630px)
   - Team/teacher photo
   - "Learn Spanish with Native Speakers" text

3. **pricing.jpg** (1200x630px)
   - Pricing highlight
   - "Flexible Plans for Every Learner" text

## Design Guidelines

- Dimensions: 1200x630px (Facebook/LinkedIn standard)
- Format: JPG (optimized, < 200KB)
- Color palette: Spanish Red (#B91C1C), Gold (#D97706), Warm neutrals
- Typography: Playfair Display for headings, Inter for body
- Logo: Include Spanish Class logo in top corner
- Ensure text is readable at small sizes (mobile preview)

## Tools

- Canva: Use "Facebook Post" template (resize to 1200x630)
- Figma: Create artboard with exact dimensions
- Photoshop: Save for web (JPG, quality 80)
